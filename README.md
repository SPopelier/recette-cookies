# 🍪 Exercice Cookie – Premiers pas avec le terminal

## 🎯 Objectif

Cet exercice a été réalisé dans le cadre de ma formation pour **apprendre les bases de l'utilisation du terminal** (ligne de commande).  
Plutôt que de simplement taper des commandes dans le vide, nous avons choisi de suivre… une **recette de cookies**.

Oui, des cookies. Parce que coder c’est bien, mais coder avec l’odeur de pâte sablée, c’est mieux.

---

## 📚 Compétences travaillées

- Se déplacer dans les dossiers avec `cd`
- Créer des fichiers avec `touch`
- Créer des dossiers avec `mkdir`
- Lister le contenu d’un dossier avec `ls`
- Éditer un fichier avec `nano` ou `vim` (selon les goûts)
- Supprimer fichiers et dossiers (`rm`, `rmdir`)
- Utiliser les permissions (`chmod`, `chown`)
- Copier, déplacer, renommer (`cp`, `mv`)
